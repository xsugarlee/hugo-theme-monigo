**MoniGo** 是一个为技术博客设计的极简主题，以等宽字体为核心，专注于内容阅读与代码展示体验。

## 设计理念

主题采用 **Maple Mono** 作为英文字体，**苹方-简** 作为中文字体，营造统一的等宽字体视觉风格。整体配色以浅灰色 `#E9E4E0` 为主色调，搭配深色文字，呈现简洁干净的阅读界面。

## 核心功能

### 文章系统

- 支持 Page Bundle 格式，文章可包含独立的配图资源
- 自动加载文章缩略图，显示在标题下方
- 支持文章标签分类
- 分页浏览，每页 10 篇文章
- 上一篇 / 下一篇导航

### 代码展示

- 浅色主题代码高亮，背景色 `#eef0f2`
- 自动识别代码语言并显示标签
- 一键复制代码功能
- 支持语法高亮：关键字、字符串、注释、函数等
- 隐藏滚动条，保持界面整洁

### 视觉设计

- 响应式布局，适配桌面端与移动端
- 深色模式支持（自动跟随系统设置）
- 回到顶部按钮，滚动超过 300px 自动显示
- 无边框设计，减少视觉干扰

### 首页展示

- 显示最近 8 篇文章
- 文章描述支持 Markdown 格式渲染
- 「查看全部文章」链接跳转完整列表

## 技术实现

| 技术 | 说明 |
|------|------|
| Hugo | 静态网站生成器 |
| Page Bundle | 文章资源管理 |
| Chroma | 代码语法高亮 |
| CSS Variables | 主题配色系统 |
| Vanilla JS | 交互功能实现 |
